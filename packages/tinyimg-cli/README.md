# tinyimg-cli

CLI tool for TinyPNG image compression.

## Installation

```bash
npm install -g tinyimg-cli
```

## Usage

```bash
# Compress images
tinyimg *.png

# Manage API keys
tinyimg key add <key>
tinyimg key list
```

## Features

- Batch compression with glob patterns
- Interactive API key management
- Progress tracking and reporting
- Smart caching
