# tinyimg-unplugin

unplugin for automatic image compression during build.

## Supported Tools

- Vite
- Webpack
- Rolldown

## Installation

```bash
npm install tinyimg-unplugin
```

## Usage

```javascript
// vite.config.js
import tinyimg from 'tinyimg-unplugin'

export default {
  plugins: [tinyimg()]
}
```
